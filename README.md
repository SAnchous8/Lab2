# Задание 1

### Текст задачи

Получить список из минимальных цифр натуральных чисел, содержащихся в
исходном списке

### Алгоритм решения
Алгоритм решения

1. Попадаем в функцию main

2. Запрашиваем у пользователя количество чисел:

   Выводим сообщение: "Сколько чисел будете вводить?"

   Считываем введенное значение в значение count

3. Создаем список numbers:

   Запускаем цикл for от 1 до count

   Для каждого значения i вызываем функцию readNumber i

   Добавляем полученное значение в список

4. В функции readNumber с параметром index:

   Выводим сообщение: "Число {index}: "

   Считываем введенное значение в значение input

   Проверяем корректность ввода:

   Если input <= 0, выводим сообщение "Ошибка, число должно быть натуральным" и возвращаем -1

   Если input > 0, возвращаем input

5. Возвращаемся в функцию main:

   Выводим сообщение: "Исходный список:"

   Выводим содержимое списка numbers

6. Для каждого значения в списке numbers вызываем функцию minDigit

7. В функции minDigit с параметром n:

   Проверяем, равно ли n нулю:
   
   Если n = 0, возвращаем 0
   
   Если n > 0, вызываем функцию getDigits с аргументами n и начальным минимумом 9

8. В функции getDigits с параметрами num и currentMin:

   Проверяем базовый случай:
   
   Если num = 0, возвращаем currentMin
   
   Если num > 0, извлекаем последнюю цифру digit из num
   
   Вызываем getDigits с num/10 и обновленным минимумом min digit currentMin

9. Возвращаемся в функцию main:

   Выводим сообщение: "Минимальные цифры:"

   Выводим список результатов, полученных от функции minDigit

10. Завершаем программу с кодом возврата 0

# Задание 2

### Текст задачи

Найти количество элементов списка, в которых встречается заданная цифра.

### Алгоритм решения

Алгоритм решения

1. Попадаем в функцию main

2. Запрашиваем у пользователя количество чисел:

   Выводим сообщение: "Сколько чисел будете вводить?"

   Считываем введенное значение в значение count

3. Создаем список numbers:

   Запускаем цикл for от 1 до count

   Для каждого значения i вызываем функцию readNumber i

   Добавляем полученное значение в список

4. В функции readNumber с параметром index:

   Выводим сообщение: "Число {index}: "

   Считываем введенное значение и возвращаем его

5. Возвращаемся в функцию main:

   Выводим сообщение: "Исходный список:"

   Выводим содержимое списка numbers

6. Запрашиваем у пользователя искомую цифру:

   Выводим сообщение: "Какую цифру ищем? "
   
   Считываем введенное значение в значение targetDigit

7. Проверяем корректность ввода:

   Если targetDigit меньше 0 или больше 9, выводим сообщение "Ошибка: цифра должна быть от 0 до 9"
   
   Если targetDigit от 0 до 9, вызываем функцию countElementsWithDigit с аргументами numbers и targetDigit
   
   Сохраняем результат в значение result
   
   Выводим сообщение: "Количество элементов, содержащих цифру {targetDigit}: {result}"

8. В функции countElementsWithDigit с параметрами numbers и targetDigit:

   Используем List.fold для подсчета элементов
   
   Начальное значение аккумулятора равно 0
   
   Для каждого значения number в списке numbers вызываем функцию containsDigit с аргументами number и targetDigit
   
   Добавляем результат вызова containsDigit к аккумулятору
   
   Возвращаем итоговое значение аккумулятора

9. В функции containsDigit с параметрами number и digit:

   Проверяем, равно ли number нулю:
   
   Если number = 0 и digit = 0, возвращаем 1
   
   Если number = 0 и digit ≠ 0, возвращаем 0
   
   Если number > 0, вызываем функцию checkDigits с аргументом number

10. В функции checkDigits с параметром num:

   Проверяем базовый случай:

   Если num = 0, возвращаем 0
   
   Если num > 0, извлекаем последнюю цифру currentDigit из num
   
   Проверяем, равна ли currentDigit искомой цифре digit:
   
   Если равна, возвращаем 1
   
   Если не равна, вызываем checkDigits с num/10

11. Завершаем программу с кодом возврата 0
