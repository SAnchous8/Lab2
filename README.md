# Задание 1

### Текст задачи

Получить список из минимальных цифр натуральных чисел, содержащихся в
исходном списке

### Алгоритм решения

1. Попадаем в функцию main

2. Запрашиваем у пользователя количество чисел:

   2.1. Выводим сообщение: "Сколько чисел будете вводить?"
   2.2. Считываем введенное значение в значение count

3. Создаем список numbers с помощью генератора списка:
   Для каждого индекса i от 1 до count:

      3.1. Вызываем функцию readNumber с аргументом i для чтения числа
      3.2. Добавляем полученное значение в список

4. В функции readNumber с параметром index:
   4.1. Выводим сообщение: "Число {index}: "
   4.2. Считываем введенное значение в значение input
   4.3. Проверяем корректность ввода:
       4.3.1. Если input <= 0 (значение не является натуральным):
              4.3.1.1. Выводим сообщение "Ошибка, число должно быть натуральным"
              4.3.1.2. Возвращаем -1 как признак ошибки
       4.3.2. Иначе:
              4.3.2.1. Возвращаем введенное значение

5. Возвращаемся в функцию main:
   5.1. Выводим сообщение: "\nИсходный список: "
   5.2. Выводим содержимое списка numbers

6. Для каждого значения в списке numbers:
   6.1. Вызываем функцию minDigit с аргументом (текущее значение)

7. В функции minDigit с параметром n:
   7.1. Проверяем особый случай:
        7.1.1. Если n = 0:
               7.1.1.1. Возвращаем 0 (минимальная цифра значения 0)
        7.1.2. Иначе:
               7.1.2.1. Вызываем вспомогательную рекурсивную функцию getDigits с аргументами: n и начальным минимумом 9

8. В функции getDigits с параметрами num и currentMin:
   8.1. Проверяем базовый случай:
        8.1.1. Если num = 0 (все цифры обработаны):
               8.1.1.1. Возвращаем текущий минимум currentMin
   8.2. Иначе (рекурсивный случай):
        8.2.1. Извлекаем последнюю цифру значения: digit = num % 10
        8.2.2. Рекурсивно вызываем getDigits с:
               8.2.2.1. новым значением: num / 10 (отбрасываем последнюю цифру)
               8.2.2.2. обновленным минимумом: min digit currentMin (берем минимальное из текущей цифры и накопленного минимума)

9. Возвращаемся в функцию main:
   9.1. Выводим сообщение: "Минимальные цифры: "
   9.2. Выводим список результатов, полученных от применения minDigit к каждому значению

10. Завершаем программу с кодом возврата 0

Найти количество элементов списка, в которых встречается заданная цифра.

### Алгоритм решения

Алгоритм решения

1. Попадаем в функцию main

2. Запрашиваем у пользователя количество чисел:
   2.1. Выводим сообщение: "Сколько чисел будете вводить?"
   2.2. Считываем введенное значение в значение count

3. Создаем список numbers с помощью генератора списка:
   Для каждого индекса i от 1 до count:
      3.1. Вызываем функцию readNumber с аргументом i для чтения числа
      3.2. Добавляем полученное значение в список

4. В функции readNumber с параметром index:
   4.1. Выводим сообщение: "Число {index}: "
   4.2. Считываем введенное значение и возвращаем его

5. Возвращаемся в функцию main:
   5.1. Выводим сообщение: "\nИсходный список: "
   5.2. Выводим содержимое списка numbers

6. Запрашиваем у пользователя искомую цифру:
   6.1. Выводим сообщение: "Какую цифру ищем? "
   6.2. Считываем введенное значение в значение targetDigit

7. Проверяем корректность введенной цифры:
   7.1. Если targetDigit находится в диапазоне от 0 до 9:
        7.1.1. Вызываем функцию countElementsWithDigit с аргументами: список numbers и targetDigit
        7.1.2. Полученный результат сохраняем в значение result
        7.1.3. Выводим сообщение: "Количество элементов, содержащих цифру {targetDigit}: {result}"
   7.2. Иначе:
        7.2.1. Выводим сообщение: "Ошибка: цифра должна быть от 0 до 9"

8. В функции countElementsWithDigit с параметрами numbers и targetDigit:
   8.1. Используем функцию List.fold для подсчета элементов:
        Начальное значение аккумулятора: 0
        Для каждого значения number в списке numbers:
           8.1.1. Вызываем функцию containsDigit с аргументами: number и targetDigit
           8.1.2. Добавляем полученное значение (0 или 1) к аккумулятору
   8.2. Возвращаем итоговое значение аккумулятора

9. В функции containsDigit с параметрами number и digit:
   9.1. Проверяем особый случай:
        9.1.1. Если number = 0:
               9.1.1.1. Если digit = 0, возвращаем 1 (число 0 содержит цифру 0)
               9.1.1.2. Иначе возвращаем 0
   9.2. Иначе:
        9.2.1. Вызываем вспомогательную рекурсивную функцию checkDigits с аргументом number

10. В функции checkDigits с параметром num:
    10.1. Проверяем базовый случай:
         10.1.1. Если num = 0 (все цифры проверены, цифра не найдена):
                10.1.1.1. Возвращаем 0
    10.2. Иначе (рекурсивный случай):
         10.2.1. Извлекаем последнюю цифру значения: currentDigit = num % 10
         10.2.2. Проверяем, совпадает ли текущая цифра с искомой:
                10.2.2.1. Если currentDigit = digit:
                          10.2.2.1.1. Возвращаем 1 (цифра найдена)
                10.2.2.2. Иначе:
                          10.2.2.2.1. Рекурсивно вызываем checkDigits с новым значением: num / 10 (отбрасываем последнюю цифру)

11. Завершаем программу с кодом возврата 0
